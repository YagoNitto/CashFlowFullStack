# Sistema de Gerenciamento Financeiro

## Visão Geral

Este projeto é um sistema de gerenciamento financeiro que permite aos usuários gerenciar suas finanças pessoais. O sistema inclui funcionalidades completas de CRUD (Create, Read, Update, Delete) para categorias e transações, oferecendo uma interface intuitiva para a organização e acompanhamento de despesas e receitas.

## Funcionalidades

- [x] **CRUD de Categorias**
  - [x] Criação de novas categorias.
  - [x] Visualização de categorias existentes.
  - [x] Edição de categorias.
  - [x] Exclusão de categorias.

- [x] **CRUD de Transações**
  - [x] Registro de novas transações (despesas e receitas).
  - [x] Visualização de transações existentes.
  - [x] Edição de transações.
  - [x] Exclusão de transações.
  - [x] Busca de transações por período. 

- [x] **Sistema de login**
  - [x] Cadastro de usuário.
  - [x] Diferentes níveis de permissões.

## Requisitos

- **Back-end**: ASP.NET Core
- **Front-end**: Blazor Web Assembly, MudBlazor
- **Banco de Dados**: SqlServer
